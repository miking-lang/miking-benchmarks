# Phylogenetic birth-death models in Birch

## Installation

To build, use:

    birch build --enable-release

To install system wide, use:

    birch install

## Run

Example:

    BIRCH_MODE="release" birch infer --tree anatinae --model crbd --output output/anatinae_crbd.json
