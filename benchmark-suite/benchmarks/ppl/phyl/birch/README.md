# Phylogenetic birth-death models in Birch

## Installation

To build, use:

    birch build --enable-release

To install system wide, use:

    birch install

## Run

Example:

    BIRCH_MODE="release" birch infer --config experiment1 --tree alcedinidae --nparticles 10000
